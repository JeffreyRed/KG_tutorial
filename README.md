# Knowledge Graph Tutorial

A step-by-step research-oriented tutorial for building a Knowledge Graph pipeline in Python.

This repository demonstrates how to:

1. Extract entities from text
2. Build a knowledge graph
3. Train knowledge graph embeddings
4. Query and analyze the graph
5. Extend the system with modern AI techniques

The project gradually introduces advanced topics such as RDF ontologies, Neo4j graph databases, embedding models, and GraphRAG.

---

# Project Goals

This project helps you understand the **theory and practice of Knowledge Graphs**.

By the end you will learn how to:

- Represent knowledge as triples
- Construct knowledge graphs from text
- Store graphs in graph databases
- Train embedding models like TransE and RotatE
- Combine knowledge graphs with LLMs
